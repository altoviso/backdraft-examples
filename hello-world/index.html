Backdraft Hello World Example Application

This application defines a "HelloWorld" Backdraft Component. The component prints "Hello, World" in several languages as given by the language property of the component. The main program creates and renders an instance of HelloWorld in the black below, and then changes the language property every second, causing the component to lookup and reflect the translation to the DOM.

The main program creates second HelloWorld component in the red box. A reference to this second component is saved at window.helloWorld. You can open the debugger and set the language component explicitly (e.g., window.helloWorld.langauge = "French") and see the component update the display.
